# zine-sample

# ✍️ ZINE制作プロジェクト

このリポジトリは、ZINEの原稿執筆と進行管理を行うためのものです。  
執筆・編集に関わるメンバーが、効率よくコラボレーションできるように構成されています。

---

## ✅ このリポジトリでやること

- 原稿アイデアの共有（→ [Issues](../../issues) を使います）
- 原稿の執筆・編集（→ [articles/](./articles) フォルダにMarkdownで書きます）
- 原稿の履歴管理（→ GitHubでバージョン管理）
- 編集状況の見える化（→ Issueのラベルやステータス）

---
